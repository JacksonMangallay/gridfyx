# Introduction

Gridfyx is a progressive PHP Web Framework that focuses on helping you build solid web applications without compromising your learning experience. This tool guides you on how to manage your application effectively but let's you build the very foundation of your application freely.

Gridfyx lets you learn the very core of PHP language by not abstracting almost everything just like any other frameworks out there. This means that while you are developing, you are learning PHP core effecively and being guided by Gridfyx.

Gridfyx is for people who wants to learn core PHP but want some guidance. Also, Gridfyx fits well for people who want to use a PHP framework but don't want very bloated one.

# Compatibility

Fow now, Gridfyx only runs on Apache server. Support for Nginx and Microsoft IIS servers are still in works.

Apache mod_rewrite must be enabled before installing Gridfyx.

Gridfyx supports PHP 7.0 version or newer.

Latest stable version: 1.4

# Installation

Gridfyx installation is very easy. It just takes 3 steps:

  • 1. Download Gridfyx PHP Framework.
  • 2. Unzip the package.
  • 3. Upload the files to your server.

And you're done. All you have to do now is visit your domain and see Gridfyx live in action.
